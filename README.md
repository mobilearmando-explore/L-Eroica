# L’Eroica Ride Calendar

A simplified calendar of L’Eroica and Nova Eroica rides, with typical seasonal weather for each location.

## Publish with GitHub Pages

1. Upload `index.html` and `.nojekyll` to the repository root.
2. Open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/ (root)`, then save.

The project site should appear at:

`https://<github-username>.github.io/L-Eroica/`
